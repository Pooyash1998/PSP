Detailed Description

Contains the third layer to interpret keyboard input. Contains functionality to register handlers for certain key types and events.

◆ keyb_input_process()

void keyb_input_process ( uint8_t  input)

This function takes unprocessed bytes from layer 2 and parses them into the layer 3 key buffer. Uses the translate functions from keyb_scancode.h to get key and key type for an input byte and updates the key buffer. Also sets the active modifiers. If the translations yielded a valid result, hands off to keyb_char_modifiers for further processing of the key.

Parameters
inputThe unprocessed byte from layer 2.
