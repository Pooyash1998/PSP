◆ keyb_shiftChar()

int8_t keyb_shiftChar ( uint8_t  character)

A function that shifts the passed char. Applying this function once will modify the passed value like the SHIFT-key would. Applying this function another time will return a value that would be expected if Alt Graph was pressed.

Parameters
characterThe (class 1) character to shift.
Returns
the character after shifting it.

◆ keyb_unNumChar()

void keyb_unNumChar ( Key *  keyBuffer)

A function to modify the value of the numpad. The default value from the numpad keys will be the value you would expect if the NUM mode is ON. After applying this function the numpad key will have the meaning as if the NUM mode were OFF.

Parameters
keyBufferPointer to the key that should be modified. Make sure type and key are set correctly.
